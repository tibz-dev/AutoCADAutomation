AutoCAD Automation System (C#)

This project is a C#-based automation tool that generates AutoCAD drawings dynamically using user input. It supports both electrical design and civil floor planning, making it useful for students and early-stage engineers.

Features

Electrical Mode

Room layout generation
Automatic placement of lights and switches
Smart wiring logic grouped per switch
Distribution board and main supply connections
Standard symbols for lights and switches
Automatic labeling (L1, S1, DB)

Civil Mode

Multi-room floor plan generation
Wall thickness (double wall design)
Door placement with user-defined position
Window generation
Basic furniture placement (bed, table, toilet)
How It Works

The system uses C# with AutoCAD COM automation to:

Accept user input such as dimensions and components
Generate drawing geometry programmatically
Create and save a DWG AutoCAD file
Technologies Used
C# (.NET)
AutoCAD COM (ActiveX Automation)
Visual Studio Code
Use Cases
Learning CAD automation
Rapid layout prototyping
